The |curveconvert| program
==========================

.. autosummary::
   :toctree: generated

The |curveconvert| program converts data files from other curve tracers to |pypsucurvetrace| format. This allows using these data with the |pypsucurvetrace| tools.

Running |curveconvert|
----------------------
The |curveconvert| program is invoked from the command line. Full documentation is not yet available in this document, but can be accessed from the |curveconvert| program directly:

.. code-block:: console

   curveconvert --help
